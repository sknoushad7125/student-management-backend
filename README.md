# Student Management System (Backend)

A backend REST API built using Spring Boot for managing students.

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven

## Features
- Add student
- View all students
- Update student
- Delete student

## API Endpoints
- POST /students
- GET /students
- PUT /students/{id}
- DELETE /students/{id}

## How to Run
1. Clone the repository
2. Configure MySQL in application.properties
3. Run the Spring Boot application
4. Test APIs using Postman
